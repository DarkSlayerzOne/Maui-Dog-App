# Maui-Dog-App

A Sample MAUI App that consumes a dog service web api.


## Requirements
 - Visual Studio 2022 Preview
 - .NET 6 SDK
 - Docker Desktop


 ## Run the Services

 ### Build the image
 
 ```
docker build -t dog-services .
 ```

 ### Run Services
 
 ```
 docker compose up -d
 ```
